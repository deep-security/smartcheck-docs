Deep Security Smart Check documentation

Deep Security Smart Check has two types of documentation:

  • The Smart Check help (now part of the Trend Micro Cloud One - Container Security help) describes how Smart Check works and provides installation and configuration instructions

    Note: If you have a Smart Check license but haven't subscribed to Container Security, you can continue to use Smart Check on its own and disregard the parts of the help pertaining to the Container Security service. You can also register for a free trial of Container Security.

  • The API docs provide information on using the Smart Check API.
